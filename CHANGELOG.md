# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-08

### Added
- Offline-first personal Kanban task manager with local SQLite database.
- Drag-and-drop task movement between columns (To Do, In Progress, Done) and reordering within columns.
- Workspace Analytics dashboard featuring status pie chart and priority bar chart.
- Custom automated rules engine (Butler rules) for auto-tagging, auto-prioritizing, and appending metadata.
- Sleek modern UI design with Syntactic Management color palette (dark/light themes).
- Beautiful custom styled Delete Project confirmation dialog.
- Cross-platform desktop (macOS/Windows) and mobile (Android/iOS) build configurations.
- Onboarding Showcase Board with rich markdown support guides.
